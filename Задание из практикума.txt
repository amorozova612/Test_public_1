	1.	В Slack или другим способом (в личном общении) найдите партнёра для выполнения этого задания. Договоритесь о том, как, когда, в какие сроки будете его выполнять и обсуждать. 
	2.	Дайте ссылку на свой репозиторий своему коллеге. Пусть он создаст у себя локально клон вашего репозитория. Репозиторий должен быть публичным!
	3.	Коллега добавляет файл в свой локальный, клонированный с вашего, репозиторий.
	4.	Коллега, используя приложение Git GUI, добавляет файл в индекс, делает коммит и отправляет изменения на сервер (то есть в ваш репозиторий).
	5.	Проверьте, что файл, добавленный коллегой, появился в вашем репозитории на GitHub. С помощью приложения Git GUI добавьте изменения в свой локальный репозиторий.
	6.	Теперь аналогичный процесс повторите вы со своей стороны: добавьте файл в свой локальный репозиторий. С помощью приложения GitGUI добавьте файл под версионный контроль и отправьте изменения на сервер.
	7.	Ваш коллега получает изменения с вашего репозитория на свой компьютер, используя Git GUI, и подтверждает, что всё ОК!
